# FlightSurety

FlightSurety is a sample application project for Udacity's Blockchain course.

## Install

This repository contains Smart Contract code in Solidity (using Truffle), tests (also using Truffle), dApp scaffolding (Angular) and server app scaffolding.

To install, download or clone the repo, then:

Step 1: Server
`npm install`
`truffle compile`
Step 2: dApp
`cd dapp2`
`npm install`

## Develop Client

To run truffle tests:

`truffle test ./test/flightSuretyData.js`
`truffle test ./test/oracles.js`

To use the dapp:

`truffle migrate`
`npm run dapp`

Hint: Don't forget to install packages of angular app

To view dapp:

`http://localhost:8000`

## Develop Server

`npm run server`
`truffle test ./test/oracles.js`

## Deploy

To build dapp for prod:
`npm run dapp:prod`

Deploy the contents of the ./dapp folder
